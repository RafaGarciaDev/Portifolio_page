## COMANDOS PARA SUBIR:
```bash
# Verificar status
git status

# Adicionar arquivos
git add index.html certificados.html projetos.html

# Fazer commit (escolha uma das opções acima)
git commit -m "sua mensagem aqui"

# Subir para o repositório
git push origin main
# OU
git push origin master
```

## DICA - Ver diferenças antes de commitar:
```bash
git diff index.html
git diff certificados.html

```
